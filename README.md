# WildSphere 🌍

A comprehensive platform dedicated to wildlife research, education, and conservation campaigns. WildSphere empowers researchers, educators, and conservation enthusiasts to collaborate, share knowledge, and drive meaningful environmental impact.

## 🎯 Mission

To create an accessible, collaborative ecosystem where wildlife research meets education and conservation action, enabling communities to protect biodiversity and promote sustainable practices.

## ✨ Features

- **Wildlife Research Hub** - Share and access research data, findings, and publications
- **Educational Resources** - Curated learning materials and courses on wildlife conservation
- **Campaign Management** - Organize and track conservation campaigns and initiatives
- **Community Collaboration** - Connect with researchers, educators, and conservation professionals
- **Data Visualization** - Interactive tools to explore wildlife data and conservation metrics
- **Resource Library** - Access to conservation tools, guides, and best practices

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/kamunyuraphael/WildSphere.git
cd WildSphere
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`
